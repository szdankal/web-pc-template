### HTML 组件

> 设计原理

- 用于解决富文本导出之后样式不一致的问题，配置同等的 css 样式来完成该问题的解决

> 使用样例

```html
<dankal-html :html="html" />
```

> 组件说明

- 组件属性

  | 属性  |  类型  |   默认   | 传入（require） | 说明                                                           |
  | :---- | :----: | :------: | :-------------: | :------------------------------------------------------------- |
  | html  | string |    ''    |      true       | 基于 v-model 实现的数据双向绑定                                |
  | width | string | '1024px' |      false      | 用于设置文档的宽度，注意这个是必须的否则会存在排版不一致的问题 |
