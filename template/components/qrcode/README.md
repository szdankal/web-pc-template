### QRCODE 组件

> 设计原理

- 基于 qrcode 生成二维码

> 使用样例

```html
<dankal-qrcode data="linksystem" />
```

> 组件说明

- 组件属性

  | 属性   |  类型  | 默认 | 传入（require） | 说明                   |
  | :----- | :----: | :--: | :-------------: | :--------------------- |
  | data   | string |  ''  |      false      | 用于转换成二维码的内容 |
  | config | object |      |      false      | 用于配置二维码的生成   |
